## An Investigation of *Physalis acutifolia* Incompatibility System  

This folder is in part funded by the Graduate Research Internship Program (GRIP) NSF program.
This is a collaboration with Pat Reeves, Ann Reilley, Chris Richards, and Stacey Smith.  

### Outline of the Project  
This project will look at the different genes that will are expressed in self-compatible and 
self-incompatible individuals.  

### Folder Contents  
**Sandbox** This folder contains practice work.
